Ejercicio 1

HTTP
HyperText Transfer Protocol (Protocolo de Transferencia de Hipertexto). Es el protocolo de comunicación utilizado para la transferencia de información en la World Wide Web. Define cómo los mensajes son formateados y transmitidos, y cómo los servidores y navegadores deben responder a diversas comandos.
HTML
HyperText Markup Language (Lenguaje de Marcado de Hipertexto). Es el lenguaje estándar utilizado para crear y diseñar páginas web. Utiliza etiquetas para estructurar el contenido, como texto, imágenes, y otros medios, en una página web.
URL
Uniform Resource Locator (Localizador Uniforme de Recursos). Es la dirección utilizada para acceder a recursos en la web, como páginas web, imágenes, y otros archivos. Una URL típica incluye el protocolo (como HTTP o HTTPS), el nombre del dominio, y la ruta al recurso.
TCP/IP
Transmission Control Protocol/Internet Protocol (Protocolo de Control de Transmisión/Protocolo de Internet). Es el conjunto de protocolos de comunicación utilizados para interconectar dispositivos en Internet. TCP/IP especifica cómo los datos son empaquetados, direccionados, transmitidos, enrutados, y recibidos.
Internet
Una red global de redes interconectadas que utilizan el protocolo TCP/IP para comunicarse. Internet permite la transmisión de datos y el acceso a una vasta cantidad de recursos y servicios, como la World Wide Web, correo electrónico, y transferencia de archivos.
W3C
World Wide Web Consortium (Consorcio de la World Wide Web). Es una organización internacional que desarrolla estándares y guías para asegurar el crecimiento a largo plazo de la Web. El W3C es responsable de estándares como HTML, CSS, y XML.
Mozilla Foundation
Una organización sin fines de lucro dedicada a la creación de software de código abierto, incluyendo el navegador web Firefox. La fundación promueve la innovación, la apertura, y la participación en Internet, y trabaja para mantener la web como un recurso público y accesible para todos.
